# images-vue

## Introduction

An app that allow people to upload, view, delete images. Written in Vue.Js
Demo: http://anhdt-images.herokuapp.com/

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```
